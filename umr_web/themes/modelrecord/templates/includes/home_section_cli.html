Create your own Unifed Model Record

The tool can be used to generate the following outputs: LaTeX, PDF, HTML and GraphViz

  1. Clone the example repo $ git clone git@github.com:modelrecords/exampleusage.git
  2. Create a conda env $ conda env create -f environment.yml
  3. Activate conda env $ conda activate modelrecords_exampleusage
  4. Profit
